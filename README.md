# News Site — Тестове завдання

Базовий сайт новин на PHP + MySQL без фреймворків.

## Структура файлів

| Файл | Опис |
|---|---|
| `index.php` | Головна сторінка — список новин, пагінація, пошук, фільтр за датою |
| `news.php` | Повна сторінка окремої новини |
| `admin.php` | Адмінка — список новин із видаленням |
| `admin_form.php` | Форма створення / редагування новини |
| `db.php` | PDO-підключення до БД |
| `config.php` | Налаштування підключення (не в репо — створюється з `config.example.php`) |
| `config.example.php` | Шаблон конфігурації |
| `style.css` | Базові стилі |
| `schema.sql` | SQL-схема бази даних із тестовими даними |
| `Dockerfile` | PHP 8.2 + Apache + pdo_mysql |
| `docker-compose.yml` | PHP app + MySQL 8 |

## Запуск через Docker (рекомендовано)

Потрібен тільки [Docker Desktop](https://www.docker.com/products/docker-desktop/).

```bash
git clone https://github.com/uaeconomclass/hlyanets-news.git
cd hlyanets-news
docker-compose up -d
```

Відкрити у браузері: [http://localhost:8081](http://localhost:8081)

Зупинити:

```bash
docker-compose down
```

> База даних ініціалізується автоматично зі `schema.sql` при першому запуску.  
> Якщо потрібно скинути дані: `docker-compose down -v && docker-compose up -d`

## Запуск локально (без Docker)

### 1. Вимоги

- PHP 8.0+
- MySQL 8.0+

### 2. Налаштувати базу даних

```bash
mysql -u root -p < schema.sql
```

Або відкрити `schema.sql` у phpMyAdmin та виконати.

### 3. Створити `config.php`

```bash
cp config.example.php config.php
```

Відредагувати `config.php` — вказати свої дані підключення:

```php
define('DB_HOST', '127.0.0.1');
define('DB_PASS', '');  // ваш пароль MySQL
```

### 4. Запустити вбудований PHP-сервер

```bash
php -S localhost:8080
```

Відкрити: [http://localhost:8081](http://localhost:8081)

## Функціонал

### Головна сторінка (`/`)
- Список новин: заголовок, дата, короткий опис
- Пагінація — 5 новин на сторінку
- Пошук за ключовими словами (заголовок + короткий опис)
- Фільтр за діапазоном дат
- Посилання "Детальніше" на повну сторінку новини

### Сторінка новини (`/news.php?id=N`)
- Повний текст новини
- 404 при невалідному ID

### Адмінка (`/admin.php`)
- Список усіх новин із датою
- Редагування новини
- Видалення з підтвердженням (`confirm()`)
- Кнопка "+ Додати новину"

### Форма новини (`/admin_form.php`)
- Поля: заголовок, короткий опис, текст новини
- Серверна валідація:
  - Усі поля обов'язкові
  - Заголовок — не більше 255 символів
- Режими: створення та редагування (через `?id=N`)

## Безпека

- **PDO prepared statements** — захист від SQL-ін'єкцій
- **`htmlspecialchars()`** на всіх виводах — захист від XSS
- Без зовнішніх залежностей та фреймворків
